## What I Build

- Designed a **RAG pipeline** that achieved **98% grounded responses** across thousands of real queries using **LangChain, Pinecone, and ChromaDB**
- Engineered a **multi-engine document parsing system** handling **50,000+ daily requests** with fault-tolerant OCR across providers
- Fine-tuned **GPT models on domain-specific corpora** for structured schema standardization, reducing manual correction overhead
- Built a **semantic recommendation engine** aligning embeddings with user intent signals to reduce attribute mismatch errors
- Developed an **AI tutoring application** with instructor-aligned retrieval and local LLM inference using **Llama + FastAPI**
